# Healthcare Dashboard UI

## Description
A frontend dashboard built using HTML, CSS, and JavaScript that consumes a real-world API to dynamically display healthcare-related data.

## Features
- REST API integration using Fetch API
- Secure handling of authentication headers
- Dynamic rendering of patient profile and medical information
- Clean, readable, and maintainable frontend code

## Tech Stack
- HTML
- CSS
- JavaScript

## How to Run
1. Clone the repository
2. Open `index.html` in a browser
3. Ensure an active internet connection for API access
